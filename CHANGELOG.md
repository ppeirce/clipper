## 1.0.0 - 2026-03-06

Initial public release.

Highlights:

- macOS clip-review utility for `mp4`, `mov`, `h264`, `h265`, `m4v`, and `hevc`
- keyboard-first transport and clip marking
- multiple non-overlapping clip ranges per source
- clip selection, deletion, boundary edits, and undo/redo for clip-definition actions
- drag-and-drop, File menu open, and recent-file support
- export presets for `Fast H.264`, `Quality H.264`, and `Compact HEVC`
- bundled `ffmpeg` inside the release app for self-contained exports
- signed and notarized Release packaging workflow
- unit tests, UI tests, runtime tracing, and JSONL diagnostics
